# WSGI Example

A boilerplate for Flask apps with UWSGI server.

- Python v3.11.6;
- Flask v3.0.3;
- Nginx v1.26.0

## How to start 🛠

```bash
docker-compose up --build # Creates and starts Nginx reverse proxy and Flask APP with UWSGI server
```
